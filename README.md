EasyBinds — a global hotkey manager for Windows 10/11. It lets you assign any key combination to perform system actions.

The app captures global hotkeys via Win32 API and works even when minimized to the tray. You can launch programs, close the active window, run commands, open websites, show messages, and toggle system mute. Settings are saved automatically. It has dark and light themes, Russian and English languages, run at startup option, admin elevation button, and an InnoSetup installer.
