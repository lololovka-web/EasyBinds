# EasyBinds

Менеджер глобальных горячих клавиш для Windows 10/11.

## Описание

EasyBinds позволяет назначать глобальные горячие клавиши, которые работают в любом месте системы. Программа сворачивается в трей и не мешает работе.

## Возможности

- Назначение горячих клавиш на различные действия
- Запуск программ по горячей клавише
- Закрытие активного окна
- Выполнение командной строки
- Открытие веб-сайтов
- Показ сообщений
- Управление звуком (Mute/Unmute)
- Запуск с Windows
- Переключение темы (тёмная/светлая)
- Переключение языка (русский/английский)
- Сворачивание в системный трей

## Установка

### Из исходников

```
dotnet build
```

Бинарный файл: `bin\Debug\net10.0-windows\EasyBinds.exe`

### Через InnoSetup

1. Установите [Inno Setup](https://jrsoftware.org/isinfo.php)
2. Откройте файл `installer.iss`
3. Нажмите Build > Compile
4. Запустите созданный установщик

## Использование

1. Запустите `EasyBinds.exe`
2. Нажмите "Добавить"
3. Нажмите нужную комбинацию клавиш
4. Выберите действие из списка
5. Укажите параметр (путь к файлу, URL, команда и т.д.)
6. Нажмите OK

## Горячие клавиши

Программа перехватывает глобальные сочетания клавиш через Win32 API. Комбинации работают даже когда окно свёрнуто.

## Требования

- Windows 10/11
- .NET 10 Runtime (для запуска из исходников)

## Технологии

- C# / WPF (.NET 10)
- Win32 API (RegisterHotKey, PostMessage)
- System.Text.Json для сохранения настроек
